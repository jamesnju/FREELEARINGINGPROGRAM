# Free Learning Program

## Mission
Providing quality education access to underserved communities across Kenya.

## Features
- 📚 Free educational programs
- 💻 Digital skills training
- 🎓 Mentorship program
- 🤝 Community support

## Tech Stack
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (ES6+)
- Public APIs (JSONPlaceholder, Quotes API, Weather API)

## Setup Instructions
1. Clone the repository
2. Open index.html in your browser
3. Or deploy to GitHub Pages

## API Integration
- Motivational quotes from Type.fit API
- Weather updates from Open-Meteo
- Dynamic content from JSONPlaceholder
